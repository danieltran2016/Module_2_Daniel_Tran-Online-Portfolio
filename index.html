Project Name: Dinner Picker

This will be a program/javascript that my frineds, family and I can use to help decide what or where we want to eat/do when we cant figure out what we want to do. It can host a bunch of restaurants, types of food and places. You can set it to random or you can bias it towards a certain category of food or entertainment you want to do.

User story:

I want a page/app that I can enter restuarants, rank them on a scale of how much i like them. I want to be able to choose random or targeted for a selector. I can hit random and a random restaurant will come up and help me choose where or what to eat dinner or lunch. I want it to be able to target italian if that catetgory is selected etc. If possible I want it to save the selection for a given night and not repeat the selection based on a given interval.

Acceptance criteria:

A user can hit random and a choice will be selected.

A user can select a category to narrow down the results

A user can submit entries that will be saved for the choice database

A user can rank/weigh the choices towards ones they like
